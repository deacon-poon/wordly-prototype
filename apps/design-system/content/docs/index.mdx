# Wordly Design System

Welcome to the Wordly Design System documentation. This site documents the design tokens and guidelines sourced from the [Wordly Figma Design System](https://www.figma.com/design/KIjCL0Hm88Ah2xsJmfvhaZ/Wordly-Design-System).

## Foundations

- [Colors](./foundations/colors) — Primitive color scales and semantic tokens
- [Brand Blue](./foundations/colors/brand-blue) — Primary brand color
- [Accent Green](./foundations/colors/accent-green) — Accent color
- [Action Teal](./foundations/colors/action-teal) — Action/interaction color
- [Accent Orange](./foundations/colors/accent-orange) — Secondary accent
- [Gray](./foundations/colors/gray) — Neutral palette
- [Status Colors](./foundations/colors/status) — Success, warning, error, info

## Molecules

Composed components organized into 8 sections:

- [Actions](./molecules#actions) — Button, Toggle, Toggle Group
- [Forms](./molecules#forms) — Text Area, Label, Switch, Slider, Calendar, Chip Multi-selector, Radio Group, Select, Combobox, Form Field
- [Display](./molecules#display) — Avatar, Badge, Card, Skeleton, Aspect Ratio, Separator, Progress
- [Feedback](./molecules#feedback) — Alert, Toast, Sonner, Tooltip
- [Navigation](./molecules#navigation) — Tabs, Breadcrumb, Pagination, Menubar, Navigation Menu
- [Overlays](./molecules#overlays) — Alert Dialog, Dialog, Drawer, Sheet, Popover, Hover Card, Collapsible, Accordion, Context Menu, Dropdown Menu, Command
- [Data](./molecules#data) — Table, Data Table
- [Media](./molecules#media) — Carousel, Scroll Area, Resizable, Input OTP
