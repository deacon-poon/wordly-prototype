# Badge

A compact label for displaying status, categories, or counts. Use to draw attention to metadata like notification counts, tags, or workflow states.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Default | Standard badge style |
| Secondary | Muted emphasis |
| Outline | Bordered with transparent background |
| Destructive | Error or warning indicator |

## Usage Guidelines

- Use for short, glanceable information like status labels, counts, or tags
- Keep badge text concise (1-2 words or a number)
- Do not use badges for long descriptions or interactive content
- Ensure sufficient color contrast between badge background and text in both modes
