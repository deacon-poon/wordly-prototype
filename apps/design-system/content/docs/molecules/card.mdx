# Card

A bordered surface that groups related content, typically with a header, body, and optional footer. Use for content blocks like summaries, previews, or settings panels.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use to visually group related information into a scannable unit
- Include a card header with a title when the card's purpose needs context
- Keep card content focused on a single topic or action
- Avoid nesting cards within cards; use sections or separators for sub-grouping instead
