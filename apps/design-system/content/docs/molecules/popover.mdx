# Popover

A floating panel that appears next to a trigger element, typically on click. Use for rich interactive content like date pickers, color pickers, or form snippets that need to appear inline.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use for interactive content that is too complex for a tooltip but does not need a full dialog
- Position the popover relative to its trigger, adjusting for viewport edges
- Close the popover when clicking outside or pressing Escape
- Manage focus: move focus into the popover on open, return to trigger on close
