# Sonner

A stacked notification system that manages multiple toasts with smooth enter/exit animations. Use as the toast provider for applications that need queued, stacked notifications.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Default | Standard notification |
| Success | Positive confirmation |
| Error | Failure notification |
| Loading | In-progress indicator with spinner |

## Usage Guidelines

- Use as the global toast provider, typically mounted once at the app root
- Toasts stack visually so multiple notifications remain readable
- Support promise-based toasts for async operations (loading, success, error states)
- Ensure all toasts are accessible via live regions for screen reader announcement
