# Progress

A horizontal bar that fills to represent completion percentage. Use for file uploads, multi-step workflows, or any task with measurable progress.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Determinate | Shows a specific completion percentage |
| Indeterminate | Animated state when progress is unknown |

## Usage Guidelines

- Use determinate progress when the completion percentage is known
- Use indeterminate progress for tasks with unknown duration
- Provide a text label or percentage alongside the bar for clarity
- Ensure the progress bar has `role="progressbar"` with `aria-valuenow`, `aria-valuemin`, and `aria-valuemax`
