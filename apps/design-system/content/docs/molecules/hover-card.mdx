# Hover Card

A non-interactive preview card that appears when hovering over a link or element. Use to show a quick preview of linked content (user profiles, article summaries) without navigating away.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use for non-essential preview content that enhances browsing without requiring a click
- Do not put interactive elements (buttons, links, forms) inside hover cards
- Include an open delay to prevent accidental triggers during casual mouse movement
- Ensure content is also accessible via the linked page since hover cards are not available on touch devices
