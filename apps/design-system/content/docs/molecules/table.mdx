# Table

A standard HTML table with styled header, body, rows, and cells. Use for presenting structured, tabular data that users need to scan, compare, or reference.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use for data that is naturally organized in rows and columns (e.g., user lists, pricing plans)
- Include a clear header row to label each column
- Use consistent alignment: left-align text, right-align numbers
- For large datasets with sorting and filtering, use the Data Table molecule instead
