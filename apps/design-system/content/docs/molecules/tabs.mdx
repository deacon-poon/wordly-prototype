# Tabs

A set of tab triggers that switch between associated content panels. Use to organize related content into sections that share the same page without navigation.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Default | Standard tab bar |
| Disabled tab | Individual tab in non-interactive state |

## Usage Guidelines

- Use when content sections are related but users only need one at a time
- Keep tab labels short (1-3 words) and descriptive
- Do not use tabs when the content in each panel is interdependent or needs to be compared
- Ensure tab panels use `role="tabpanel"` with proper `aria-labelledby` associations
