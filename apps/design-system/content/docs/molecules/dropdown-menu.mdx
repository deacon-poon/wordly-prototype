# Dropdown Menu

A menu of actions that appears when clicking a trigger button. Use for grouped actions like "More options" menus, account menus, or action lists.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use for grouping secondary actions behind a single trigger button
- Organize items logically and separate groups with dividers
- Support submenus for deeply nested actions, but limit to one level of nesting
- Ensure full keyboard navigation: arrow keys to move, Enter to select, Escape to close
