# Slider

A draggable control for selecting a numeric value within a minimum and maximum range. Use for settings like volume, brightness, or price ranges.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Default | Single-thumb slider |
| Range | Dual-thumb slider for selecting a range |
| Disabled | Non-interactive state |

## Usage Guidelines

- Use when the exact numeric value is less important than the relative position in a range
- Pair with a visible value label or input field when precision matters
- Provide a sensible step interval to avoid overly granular adjustments
- Ensure the thumb is large enough to be easily dragged on touch devices
