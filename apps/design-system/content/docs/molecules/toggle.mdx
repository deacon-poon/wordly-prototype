# Toggle

A two-state button that can be toggled on or off. Use for binary settings where the state should be immediately visible, such as bold/italic formatting controls or view options.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Pressed | Active/selected state |
| Unpressed | Default inactive state |
| Disabled | Non-interactive state |

## Usage Guidelines

- Use toggles for immediate-effect settings that don't require form submission
- Ensure the pressed state is visually distinct from the unpressed state
- Pair with an icon or label so the purpose is clear without relying on state alone
- Provide `aria-pressed` for screen readers to communicate the current state
