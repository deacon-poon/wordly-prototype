# Scroll Area

A scrollable container with custom-styled scrollbars that replace native browser scrollbars. Use for content areas where consistent scrollbar styling across browsers is important.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Vertical | Vertical scrolling (default) |
| Horizontal | Horizontal scrolling |

## Usage Guidelines

- Use when native scrollbar styling would break the visual design
- Preserve native scroll behavior (momentum scrolling, keyboard scrolling)
- Show scrollbars on hover and hide them when idle for a clean appearance
- Ensure the scrollable area is keyboard accessible and works with assistive technologies
