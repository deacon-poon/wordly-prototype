# Separator

A thin line that visually divides content into distinct sections. Use between groups of related items in lists, sidebars, or form sections.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Horizontal | Divides content vertically (default) |
| Vertical | Divides content horizontally |

## Usage Guidelines

- Use to create visual breathing room between content groups
- Prefer spacing alone when the separation is obvious from layout context
- Use `role="separator"` for semantic meaning; decorative separators should be `aria-hidden`
- Keep separator styling subtle so it does not compete with content
