# Text Area

A multi-line text input for collecting longer-form content such as comments, descriptions, or messages.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Default | Standard editable state |
| Focused | Active input with focus ring |
| Disabled | Non-interactive state with reduced opacity |
| Error | Invalid input with error styling |

## Usage Guidelines

- Use for inputs that typically require more than a single line of text
- Prefer a standard input for short, single-line values like names or emails
- Set a reasonable default height and allow vertical resizing when appropriate
- Always pair with a label for accessibility; use placeholder text as a hint, not a replacement for the label
