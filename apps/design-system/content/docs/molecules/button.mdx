# Button

Triggers an action or event. Use buttons for primary calls to action, form submissions, and interactive controls throughout the interface.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Primary | High-emphasis action with Brand Blue background |
| Secondary | Medium-emphasis with muted background |
| Outline | Bordered style with transparent background |
| Ghost | Minimal style without border or background |
| Destructive | Dangerous or irreversible actions |
| Disabled | Non-interactive state with reduced opacity |

## Usage Guidelines

- Use primary buttons for the single most important action on a page
- Limit to one primary button per section to maintain clear visual hierarchy
- Pair destructive buttons with a confirmation dialog for irreversible actions
- Ensure buttons have visible focus indicators for keyboard navigation
