# Alert

An inline, non-dismissible message for surfacing important information, warnings, or errors. Use for persistent messages that relate to the content on the page.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Default | Informational message |
| Destructive | Error or critical warning |

## Usage Guidelines

- Use for persistent, contextual messages that relate to nearby content
- Prefer toasts for transient notifications that auto-dismiss
- Include a clear, actionable message describing what the user should do
- Use `role="alert"` for urgent messages that should interrupt screen reader flow
