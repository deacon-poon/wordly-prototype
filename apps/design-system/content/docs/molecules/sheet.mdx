# Sheet

A panel that slides in from the left or right edge of the viewport. Use for secondary navigation, filters, settings panels, or detail views that supplement the main content.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Left | Slides in from the left edge |
| Right | Slides in from the right edge |

## Usage Guidelines

- Use for supplementary content that does not warrant a full page or modal
- Include a close button and support Escape key to dismiss
- Dim the background with an overlay to indicate modality
- Trap focus within the sheet while it is open
