# Select

A dropdown selector for choosing one option from a list. Use when the option list is long or screen space is limited compared to radio buttons.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Light mode (`mode=light`) |
| Dark | Dark mode (`mode=dark`) |
| Collapsed | Closed state showing only the trigger (`Property 1=collapsed`) |
| Expanded | Open state showing the dropdown options (`Property 1=expanded`) |

### Figma Component Properties

| Property | Values | Description |
|----------|--------|-------------|
| `Property 1` | `collapsed`, `expanded` | Open/closed state |
| `mode` | `light`, `dark` | Color mode variant |

## Usage Guidelines

- Use when the option list has more than 7 items or screen space is constrained
- Prefer radio buttons when all options should be visible simultaneously
- Display a meaningful placeholder or default selection in the collapsed state
- Ensure the expanded dropdown is keyboard navigable with arrow keys and type-ahead
