# Tooltip

A small overlay that displays supplementary text when hovering over or focusing a trigger element. Use for icon-only buttons, truncated text, or any element that benefits from additional context.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use for brief, non-essential supplementary information
- Never put critical information in a tooltip since it is not visible on touch devices by default
- Keep tooltip text to a single sentence or short phrase
- Ensure tooltips appear on keyboard focus as well as mouse hover for accessibility
