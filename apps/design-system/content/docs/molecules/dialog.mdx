# Dialog

A modal overlay that focuses user attention on a specific task or content. Use for forms, confirmations, or detail views that require temporary focus away from the main page.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use for tasks that require focused attention but do not warrant a full page
- Include a visible close button and support Escape key to dismiss
- Trap focus within the dialog while it is open
- Return focus to the trigger element when the dialog closes
