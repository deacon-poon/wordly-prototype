# Drawer

A panel that slides in from the bottom of the viewport. Use for mobile-friendly interactions like filters, action menus, or detail views that benefit from a swipe-to-dismiss gesture.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use primarily for mobile or responsive layouts where a bottom sheet is natural
- Support swipe-down gesture to dismiss on touch devices
- Include a visible drag handle at the top to signal interactivity
- Trap focus within the drawer while it is open and restore focus on close
