# Toggle Group

A set of two-state buttons grouped together, allowing single or multiple selection. Use for toolbar-style controls where users pick from a small set of related options.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Single | Only one item can be active at a time |
| Multiple | Multiple items can be active simultaneously |

## Usage Guidelines

- Use single mode when options are mutually exclusive (e.g., text alignment)
- Use multiple mode when options can be combined (e.g., text formatting: bold + italic)
- Keep the number of items small (2-5) for quick scanning
- Ensure each toggle has an accessible label or icon with `aria-label`
