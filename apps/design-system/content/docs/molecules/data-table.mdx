# Data Table

An advanced table component with built-in support for column sorting, row filtering, pagination, and row selection. Use for data-heavy views like admin panels, dashboards, and reporting interfaces.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use when users need to sort, filter, search, or paginate through structured data
- Prefer a simple table when the dataset is small and static
- Provide column header click-to-sort with clear directional indicators
- Ensure keyboard accessibility for all interactive elements (sort buttons, pagination, row selection)
