# Input OTP

A segmented input for entering one-time passwords or verification codes. Each digit gets its own input cell, providing clear visual feedback as users type their code.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use for verification codes, two-factor authentication, and PIN entry
- Auto-advance focus to the next cell as each digit is entered
- Support paste to fill all cells at once from a copied code
- Ensure each cell is part of a single logical input for screen readers via `aria-label` on the group
