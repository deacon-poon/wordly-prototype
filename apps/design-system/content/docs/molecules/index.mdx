# Molecules

Molecules are composed components built from atoms. They represent functional UI patterns such as form controls, navigation elements, dialogs, and data displays. Each molecule supports Light and Dark mode variants.

## Actions

- [Button](./molecules/button) — Trigger actions with multiple variants and sizes
- [Toggle](./molecules/toggle) — Two-state control for on/off settings
- [Toggle Group](./molecules/toggle-group) — Group of toggles for selecting one or multiple options

## Forms

- [Text Area](./molecules/text-area) — Multi-line text input
- [Label](./molecules/label) — Accessible label for form controls
- [Switch](./molecules/switch) — Binary toggle for boolean settings
- [Slider](./molecules/slider) — Range input for selecting a numeric value
- [Calendar](./molecules/calendar) — Date picker for selecting dates
- [Chip Multi-selector](./molecules/chip-multi-selector) — Multi-select input using removable chips
- [Radio Group](./molecules/radio-group) — Single selection from a set of options
- [Select](./molecules/select) — Dropdown for choosing from a list
- [Combobox](./molecules/combobox) — Searchable dropdown with autocomplete
- [Form Field](./molecules/form-field) — Composed field with label, input, description, and validation

## Display

- [Avatar](./molecules/avatar) — User or entity representation with image or initials
- [Badge](./molecules/badge) — Status indicator or label
- [Card](./molecules/card) — Contained surface for grouping related content
- [Skeleton](./molecules/skeleton) — Placeholder for loading content
- [Aspect Ratio](./molecules/aspect-ratio) — Constrain content to a fixed aspect ratio
- [Separator](./molecules/separator) — Visual divider between sections
- [Progress](./molecules/progress) — Visual indicator of completion or loading

## Feedback

- [Alert](./molecules/alert) — Inline message for important information
- [Toast](./molecules/toast) — Temporary notification that auto-dismisses
- [Sonner](./molecules/sonner) — Stacked toast notification system
- [Tooltip](./molecules/tooltip) — Contextual hint on hover or focus

## Navigation

- [Tabs](./molecules/tabs) — Switch between content sections
- [Breadcrumb](./molecules/breadcrumb) — Hierarchical path navigation
- [Pagination](./molecules/pagination) — Navigate through paged content
- [Menubar](./molecules/menubar) — Horizontal menu with dropdown submenus
- [Navigation Menu](./molecules/navigation-menu) — Site-level navigation with mega-menu support

## Overlays

- [Alert Dialog](./molecules/alert-dialog) — Confirmation dialog requiring user action
- [Dialog](./molecules/dialog) — Modal window for focused tasks
- [Drawer](./molecules/drawer) — Sliding panel from screen edge
- [Sheet](./molecules/sheet) — Side panel overlay for secondary content
- [Popover](./molecules/popover) — Floating content anchored to a trigger
- [Hover Card](./molecules/hover-card) — Preview card shown on hover
- [Collapsible](./molecules/collapsible) — Expandable/collapsible content section
- [Accordion](./molecules/accordion) — Stacked collapsible sections
- [Context Menu](./molecules/context-menu) — Right-click action menu
- [Dropdown Menu](./molecules/dropdown-menu) — Action menu triggered by a button
- [Command](./molecules/command) — Command palette with search and keyboard navigation

## Data

- [Table](./molecules/table) — Structured data display in rows and columns
- [Data Table](./molecules/data-table) — Enhanced table with sorting, filtering, and pagination

## Media

- [Carousel](./molecules/carousel) — Scrollable content slider
- [Scroll Area](./molecules/scroll-area) — Custom scrollable container
- [Resizable](./molecules/resizable) — Drag-to-resize panels
- [Input OTP](./molecules/input-otp) — One-time password input with segmented fields
