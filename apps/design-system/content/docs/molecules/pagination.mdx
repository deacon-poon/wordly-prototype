# Pagination

A set of controls for navigating between pages of content. Use for lists, tables, or search results that span multiple pages.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use when content is divided into discrete pages (e.g., search results, data tables)
- Show the current page and provide quick access to first, last, previous, and next pages
- Use ellipsis to truncate large page ranges while keeping endpoints visible
- Ensure all pagination controls are keyboard accessible with clear focus indicators
