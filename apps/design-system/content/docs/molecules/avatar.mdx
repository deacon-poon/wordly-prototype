# Avatar

Displays a user photo, initials, or fallback icon. Use to represent people, teams, or entities across the interface.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Image | Displays a user photo |
| Fallback | Shows initials or icon when no image is available |

## Usage Guidelines

- Use in headers, comments, user lists, and anywhere identity context is helpful
- Always provide a text fallback (initials or icon) for when the image fails to load
- Include an accessible `alt` attribute describing the person or entity
- Keep sizes consistent within the same context (e.g., all avatars in a list should be the same size)
