# Skeleton

An animated placeholder that mimics the shape of content while it loads. Use to indicate that data is being fetched and reduce perceived load time.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use skeleton shapes that match the layout of the content they replace (text lines, avatars, cards)
- Prefer skeletons over spinners when the layout structure is predictable
- Avoid using skeletons for interactions that complete in under 200ms
- Ensure the animation pulse has sufficient contrast to be perceivable in both light and dark modes
