# Collapsible

A single expandable/collapsible section with a trigger that toggles visibility of its content. Use for progressive disclosure where secondary content can be hidden until needed.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Expanded | Content is visible |
| Collapsed | Content is hidden |

## Usage Guidelines

- Use for optional or secondary content that most users do not need immediately
- Clearly indicate the expanded/collapsed state with a chevron or icon
- Animate the transition smoothly for a polished experience
- Ensure the trigger is keyboard accessible and communicates state via `aria-expanded`
