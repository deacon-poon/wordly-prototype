# Context Menu

A menu that appears on right-click (or long-press on touch), offering contextual actions relevant to the target element. Use for power-user shortcuts in content-rich interfaces.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use for supplementary actions; never put critical-path actions exclusively in a context menu
- Keep menu items focused on actions relevant to the right-clicked element
- Support keyboard navigation within the menu (arrow keys, Enter to select, Escape to close)
- Include icons alongside labels for quick scanning when appropriate
