# Chip Multi-selector

A multi-select input that displays selected values as removable chips. Use for tagging, category selection, or any scenario where users pick multiple items from a set.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Light mode (`mode=light`) |
| Dark | Dark mode (`mode=dark`) |

### Figma Component Properties

| Property | Values | Description |
|----------|--------|-------------|
| `mode` | `light`, `dark` | Color mode variant |

## Usage Guidelines

- Use when users need to select multiple items and see all selections at a glance
- Each chip should have a visible remove button for easy deselection
- Provide a dropdown or autocomplete for discovering available options
- Ensure chips wrap gracefully when the container is narrow
