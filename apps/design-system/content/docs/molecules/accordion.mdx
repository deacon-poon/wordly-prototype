# Accordion

A vertical stack of collapsible sections, each with a header trigger. Use for FAQs, settings groups, or any content that benefits from showing one section at a time.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Single | Only one item open at a time |
| Multiple | Multiple items can be open simultaneously |

## Usage Guidelines

- Use when users need to scan section titles and selectively expand content
- Default to single mode to reduce cognitive load; use multiple when comparing sections
- Keep section headers concise and descriptive
- Ensure each trigger uses `aria-expanded` and controls its associated panel via `aria-controls`
