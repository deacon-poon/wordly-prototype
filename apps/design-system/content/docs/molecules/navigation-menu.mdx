# Navigation Menu

A top-level navigation component with support for link groups, mega-menu panels, and viewport-aware positioning. Use for primary site navigation with rich dropdown content.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use for primary site navigation where categories contain multiple links or content
- Keep top-level items to 5-7 for quick scanning
- Mega-menu panels should organize links into clearly labeled groups
- Ensure the full menu is keyboard navigable and closes on Escape
