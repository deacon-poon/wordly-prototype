# Command

A searchable command palette for quickly finding and executing actions. Use as a power-user feature (typically triggered by Cmd+K or Ctrl+K) for fast navigation and action execution.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use as a global search and action launcher accessible via a keyboard shortcut
- Group commands into logical categories (e.g., Navigation, Actions, Settings)
- Filter results in real-time as the user types
- Show keyboard shortcuts next to actions when available for discoverability
