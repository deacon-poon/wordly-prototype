# Calendar

A calendar view for selecting dates. Use for date pickers, scheduling interfaces, and any context where users need to choose a specific date.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Single | Select a single date |
| Range | Select a date range with start and end |
| Disabled dates | Specific dates marked as unavailable |

## Usage Guidelines

- Use when users need to select a date relative to surrounding days (e.g., booking, scheduling)
- Prefer a simple text input for well-known dates like birthdays
- Highlight today's date and the selected date clearly
- Support keyboard navigation for month/year switching and day selection
