# Aspect Ratio

A layout container that enforces a specific width-to-height ratio on its content. Use for images, videos, and embedded media to prevent layout shifts during loading.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use for media content (images, videos, maps) where maintaining proportions is critical
- Common ratios include 16:9 (video), 4:3 (photo), 1:1 (square thumbnail)
- Wrap images in aspect ratio containers to prevent cumulative layout shift (CLS)
- Pair with object-fit (cover or contain) to control how content fills the container
