# Menubar

A horizontal bar of menu triggers, each opening a dropdown submenu. Use for application-style menu systems (e.g., File, Edit, View) where actions are organized into categories.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use for desktop-style applications with many grouped actions
- Avoid for simple navigation; use a standard nav bar or tabs instead
- Support keyboard navigation: arrow keys move between menus, Enter/Space opens submenus
- Submenus should close when focus moves outside the menu bar
