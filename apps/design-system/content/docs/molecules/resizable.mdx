# Resizable

A layout component with draggable handles that let users resize adjacent panels. Use for split-view layouts like code editors, sidebars, or comparison views where users control the space allocation.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Horizontal | Panels arranged side by side |
| Vertical | Panels stacked top to bottom |

## Usage Guidelines

- Use when users need to customize how space is divided between content panels
- Provide a visible and interactive drag handle between panels
- Set minimum and maximum sizes to prevent panels from collapsing or expanding excessively
- Ensure the drag handle is keyboard accessible (e.g., arrow keys to resize)
