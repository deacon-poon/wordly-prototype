# Switch

A toggle control for switching between two states (on/off). Use for settings that take effect immediately without requiring a form submission.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| On | Active/checked state |
| Off | Inactive/unchecked state |
| Disabled | Non-interactive state |

## Usage Guidelines

- Use for settings that apply immediately (e.g., enabling notifications)
- Prefer a checkbox when the setting requires an explicit save action
- Always pair with a visible label describing what the switch controls
- Ensure the on/off states are visually distinct through color and thumb position
