# Label

An accessible text label associated with a form control. Clicking the label focuses or activates the linked input.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Required | Indicates a mandatory field |
| Optional | Standard label without required indicator |
| Disabled | Muted appearance when the associated control is disabled |

## Usage Guidelines

- Always associate labels with their form controls using `htmlFor` / `id` pairing
- Never rely on placeholder text as the sole label for a field
- Position labels above or to the left of their associated input for consistent scanning
- Use a visible required indicator (e.g., asterisk) alongside `aria-required` on the input
