# Carousel

A horizontally scrollable component for cycling through a set of items (images, cards, or content panels). Use for showcasing featured content, testimonials, or product galleries.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use for browsing a curated set of items where showing all at once is impractical
- Provide visible previous/next controls and optional dot indicators for position
- Support swipe gestures on touch devices
- Avoid auto-advancing carousels; if used, provide a pause control and respect `prefers-reduced-motion`
