# Form Field

A composed form element that groups a label, input control, description text, and validation message into a single cohesive unit. Use as the standard wrapper for all form inputs.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |
| Default | Standard field state |
| Error | Invalid state with error message displayed |
| Disabled | Non-interactive state |

## Usage Guidelines

- Use to ensure consistent spacing, alignment, and accessibility across all form inputs
- Always include a label; the description is optional for additional context
- Display validation messages below the input, styled distinctly from the description
- Associate the label, description, and error message with the input via ARIA attributes
