# Breadcrumb

A trail of links showing the user's current position within the site hierarchy. Use in content-heavy applications with deep navigation structures.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use for sites with three or more levels of navigation depth
- The last item should represent the current page and not be a link
- Use a clear separator character (e.g., `/` or `>`) between levels
- Wrap in a `nav` element with `aria-label="Breadcrumb"` for accessibility
