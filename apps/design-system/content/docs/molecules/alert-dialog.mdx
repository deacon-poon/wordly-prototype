# Alert Dialog

A modal dialog that interrupts the user to confirm a destructive or important action. Unlike a standard dialog, it requires the user to acknowledge or cancel before proceeding.

## Variants

| Variant | Description |
|---------|-------------|
| Light | Default light mode appearance |
| Dark | Dark mode appearance |

## Usage Guidelines

- Use for irreversible actions like deleting data, discarding changes, or signing out
- Always provide both a confirm and cancel action
- Write clear, specific action labels (e.g., "Delete project" not just "OK")
- Focus should be trapped within the dialog and move to the cancel button by default
