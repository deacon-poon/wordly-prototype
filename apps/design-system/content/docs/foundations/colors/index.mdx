# Colors

The Wordly color system is built on **primitive color scales** mapped through **semantic tokens**. All values are sourced from the [Wordly Figma Design System](https://www.figma.com/design/KIjCL0Hm88Ah2xsJmfvhaZ/Wordly-Design-System).

| Layer | Purpose | Example |
|-------|---------|---------|
| **Primitives** | Raw color scales (10–975) | `brand-blue-400` = `#017cff` |
| **Semantic** | Contextual usage tokens | `actions/primary` → `brand-blue-500` |
