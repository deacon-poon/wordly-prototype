# Semantic Tokens

Semantic tokens provide contextual meaning by aliasing primitive color values. Use semantic tokens instead of raw primitives — this ensures consistency and makes theme updates a single-point change.

## Dark Mode Mapping Overview

The core principle: in dark mode, **swap light backgrounds with dark values** and **swap dark foregrounds with light values** from the same scale.
