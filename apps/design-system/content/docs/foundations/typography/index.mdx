# Typography

Wordly uses **Roboto** as its primary typeface across all platforms. The type system is defined in the [Wordly Figma Design System](https://www.figma.com/design/KIjCL0Hm88Ah2xsJmfvhaZ/Wordly-Design-System?node-id=707-3095).

## Font Family

| Token | Value |
|-------|-------|
| `font/family/primary` | **Roboto** |

| Token | Size | Use case |
|-------|------|----------|
| `font/scale/100` | 8px | Micro labels |
| `font/scale/200` | 12px | Small text, captions |
| `font/scale/250` | 14px | Body text (default) |
| `font/scale/300` | 16px | Large body text |
| `font/scale/350` | 18px | Emphasized body |
| `font/scale/400` | 20px | H6 / section title |
| `font/scale/450` | 22px | Subheading |
| `font/scale/500` | 24px | H5 |
| `font/scale/600` | 32px | H4 |
| `font/scale/700` | 36px | H3 |
| `font/scale/800` | 48px | H2 |
| `font/scale/900` | 64px | H1 / Display |
| `font/scale/1000` | 80px | Hero display |
